# الروائع ERP 🚀

نظام تخطيط موارد المؤسسات المتكامل – **Rawaea ERP**

## 🎯 عن المشروع
نظام ERP سحابي عربي متكامل لإدارة:
- المبيعات والتوزيع
- المخازن والمخزون
- المشتريات والموردين
- الحسابات والمالية
- الموارد البشرية
- علاقات العملاء (CRM)
- التوصيل واللوجستيات
- المتجر الإلكتروني
