# الروائع ERP 🚀

نظام تخطيط موارد المؤسسات المتكامل – **Rawaea ERP**

## 🎯 عن المشروع
نظام ERP سحابي عربي متكامل لإدارة:
- المبيعات والتوزيع
- المخازن والمخزون
- المشتريات والموردين
- الحسابات والمالية
- الموارد البشرية
- علاقات العملاء (CRM)
- التوصيل واللوجستيات
- المتجر الإلكتروني

## 🏗️ هيكل المشروع
companies/company-1/
├── index.html ← النظام الأم
├── sales/ ← تطبيقات المبيعات (6)
├── warehouse/ ← تطبيقات المخازن (8)
├── delivery/ ← تطبيقات التوصيل (2)
├── purchasing/ ← تطبيقات المشتريات (2)
├── office/ ← تطبيقات العمليات المكتبية (5)
└── store/ ← المتجر الإلكتروني (2)

## 🛠️ التقنيات
- **Frontend:** HTML5, Tailwind CSS, Vanilla JS (SPA)
- **Backend:** Supabase (PostgreSQL + Edge Functions)
- **Hosting:** Cloudflare Pages
- **PWA:** Service Worker + Manifest

## 📦 المتطلبات
